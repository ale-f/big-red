Creating a simulation spec

At this point, your project contains all of the pieces of a complete bigraphical reactive system:

  • it has a signature, defining its controls;
  • it has an agent, defining its initial state; and
  • it has some reaction rules, specifying how the agent can be changed.

Big Red's last file format, the simulation spec, is used to group these together into a complete system, ready for execution by simulation tools.

Find the printing.bigraph-simulation-spec file in the project's root, and double-click it to open it.

As you've probably already realised, you can create a new simulation spec by choosing Big Red → Simulation spec from the New menu.

The simulation spec editor

Now that the simulation spec is complete, the Export... button in the bottom-right of the editor has become active. Use the tool menu to select a target for your simulation spec, then click Export....

The contents of the tool menu will vary depending on which plug-ins you have installed.
