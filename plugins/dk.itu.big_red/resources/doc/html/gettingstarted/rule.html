Creating a reaction rule

There are several known problems with the reaction rule editor; in particular, making changes to the redex that conflict with changes to the reactum can cause consistency problems.

Big Red can detect these problems, but it can't yet fix them — it will simply refuse to save inconsistent documents.

Reaction rules specify the ways in which bigraphical reactive systems are allowed to develop.

To create a new reaction rule, choose Big Red → Reaction rule from the New menu.

As you build the redex, you'll notice that Big Red copies your modifications into the reactum.

Big Red represents reaction rules as a redex and an edit script — a list of the changes that transform the redex into the reactum.

Although this is a slight deviation from the formal bigraphical model, it's handled transparently — Big Red automatically generates a traditional reactum when one is requested.
