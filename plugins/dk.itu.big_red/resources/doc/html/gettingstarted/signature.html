Creating a signature

When building a bigraphical reactive system, the first thing you need to do is to define a signature. Signatures contain controls, the kinds of object that make up the world you're modelling.

Find the printing.bigraph-signature file in the signatures/ folder, and double-click it to open it.

To create a new signature, choose Big Red → Signature from the New menu.

The signature editor

To add a new control to the signature, click the Add button.

You can use the Import... button to include the contents of another signature by reference.
