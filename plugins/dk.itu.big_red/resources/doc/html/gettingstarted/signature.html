Creating a signature

Before you can build a useful bigraph, you need to define a signature — a container for the types of object (the controls) that exist in your model.

When you created the project, Big Red automatically created a signature — the PLACEHOLDER.bigraph-signature file in the project's signatures/ folder. Double-click it to open it.

To create a new signature, choose Big Red → Signature from the New menu.
